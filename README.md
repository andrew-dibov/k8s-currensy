# K8s : Currensy

## Draft

- Gateway : точка входа в сервис
  - проксирует запросы по соответствующему пути
- Currency Fetcher :
  - регулярно ходит в публичное API
  - читает актуальные курсы
  - сохраняет курсы в Redis
- Currency Converter :
  - получает запрос на конвертацию
  - обращается к Redis
- Audit Logger :
  - принимает сообщения о всех конвертациях
  - складывает в clickhouse
  - сервис-потребител кафки
